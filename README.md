# The Buck-Passing Game — THJ (Résumé + Présentation + App)

Ce dépôt contient mon travail réalisé dans le cadre du module **THJ (Théorie des Jeux)** (Master 2 AMIS, UVSQ).

## Article étudié
**The Buck-Passing Game**  
Roberto Cominetti, Matteo Quattropani, Marco Scarsini

## Objectif du projet
- Comprendre le jeu **Buck-Passing**
- Présenter la dynamique stratégique et les équilibres
- Produire une synthèse structurée
- Développer une application permettant d’illustrer/simuler le jeu

## Contenu du dépôt
- `Presentation/` : support de présentation (PPTX)
- `Notes/` : notes complémentaires
- `Sources/` : Article etudier

## Fonctionnalités (app)
Exemples (à adapter selon ton app) :
- simulation d’un graphe de joueurs
- règles de passage du "buck"
- visualisation des transitions et résultats
- paramètres configurables (nombre de joueurs, stratégie, etc.)

## Compétences mobilisées
- théorie des jeux
- graphes / dynamiques stochastiques
- implémentation d’une simulation
- présentation & structuration scientifique

## Lancer l'application
L'application a été développée pour être exécutée facilement via **Google Colab**.

1. Ouvrir le notebook dans Google Colab  
2. Exécuter toutes les cellules dans l’ordre  
3. Suivre les instructions affichées dans le notebook

> Remarque : aucune installation locale n’est nécessaire.
